<!DOCTYPE>

- 它是指示浏览器关于页面使用哪个 HTML 版本进行编写的指令。 
- <!DOCTYPE> 声明必须是 HTML 文档的第一行,位于 <html> 标签之前 
- <!DOCTYPE> 声明没有结束标签,对大小写不敏感- 常用的 <!DOCTYPE> 声明类型包括,
	- HTML 5
	- HTML 4.01 Strict
	- HTML 4.01 Transitional
	- HTML 4.01 Frameset
	- XHTML 1.0 Strict
	- XHTML 1.0 Transitional
	- XHTML 1.0 Frameset
	- XHTML 1.1
